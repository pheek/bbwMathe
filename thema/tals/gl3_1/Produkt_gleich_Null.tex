\subsection{Einstiegsbeispiel}

Finden Sie die Lösungsmenge folgender Gleichnug:

$$x^4 + 7x^2 - 44 = 0$$

\TNT{6}{
  1. Faktorzerlegung
  $$(x^2 + 11)\cdot{}(x^2-4) = 0$$
  ... weiter ...
  $$(x^2 + 11)\cdot{}(x+2)\cdot{}(x-2) = 0$$

    Nun reicht es, wenn einer der Faktoren = 0 ist:

    $$\lx=\{-2; 2\}$$
  }

\subsection{Taschenrechner}
Beispiel

$$x^3 + 11x  \ge{} 70 - 10x^2$$

Verschiedene Lösungsansätze


a) Taschenrechner: \LoesungsRaumLang{\texttt{solve($..., x$)} liefert $-7 \le x \le -5 \text{ or } x\ge 2$}

\leserluft{}

b) Graphisch auch mit TR (ablesen aus Graph):

\TNTeop{\texttt{f(x) := $x^3 + 11x$} und \texttt{g(x) := $70 - 10x^2$}
  
  Lies ab, wo nun $f(x) \ge g(x)$ oder wo $f(x) - g(x) \ge 0$ \vspace{6mm}

  
  optional

  c) Die harte Tour\footnote{Solche und ähnliche Verfahren wendet auch der
  Taschenrechner an. Erinnerung an Primarschule: Schriftliches Multiplizieren 1573:13 = 121 (ist mit $x=10$ auch eine Polynomdivision).}:
  \begin{enumerate}
  \item Wie bei einer quadratischen Gleichung alles auf eine Seite
    nehmen: $$x^3 +10x^2 -11x -70 = 0$$

  \item Annähern, pröbeln  ($x=0, x=1, x=2$)
    Hier finden wir, dass $x=2$ die Ungleichung (bzw. die Gleichung) löst.

  \item Ziel: Gleichung vom Typ «Produkt gleich Null»:
    $$(x-2)\cdot(x + ???)\cdot(x + ???) = 0$$

  \item (optional vorzeigen) Polynomdivision:
    $$(x^3 + 10x^2 + 11x - 70) = (x-2) \cdot (???)$$
    $$(x^3 + 10x^2 + 11x - 70) : (x-2) = (x^2 + 12x + 35)$$

  \item Lösung: $$x^3 - 4x^2 -31x + 70 = (x-2)\cdot(x+5)\cdot(x+7)$$
\end{enumerate}

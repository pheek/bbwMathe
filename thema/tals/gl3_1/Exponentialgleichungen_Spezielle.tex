\subsubsection{$\ln$: Logarithmus Naturalis}

Geben Sie mit Hilfe des natürlichen Logarithmus $\ln()$ an:

$$7^x = 5$$
\TNT{16}{
$$\ln(7^x) = \ln(5)$$ 
$$x\cdot{}\ln(7) = \ln(5)$$ 
$$x = \frac{\ln(5)}{\ln(7)}$$ 
}%% end TNT

\subsubsection{Substitution}

Finden Sie die Lösungsmenge $\lx$:

$$2\cdot{} 5^x - 5^{2x} = 1$$

\TNT{10}{
  Substituiere $u := 5^x$

  $$2u -u^2 = 1$$
  Quadratische Gleichung:
  $$0 = u^2 -2u + 1$$
  $$0 = (u-1)(u-1)$$
  daraus folgt:
  $$u= 1$$
  Rücksubstitution:
  $$1=5^x$$
  Und daraus:
  $$x=0 ; \lx=\{0\}$$

}%% end TNT

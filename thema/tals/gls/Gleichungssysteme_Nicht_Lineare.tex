\subsubsection{Einstiegsbeispiel}
Lösen Sie die Wurzelgleichung:

$$\sqrt{x+1} = \frac{x-3}{2}$$

\TNT{12.0}{Definitionsbereich: $x > -1$.

  Beidseitig mal zwei:
  $$2\sqrt{x+1} = x-3$$ Danach quadrieren (es können Scheinlösungen auftreten):
  $$4(x+1) = (x-3)^2$$
  Quadratische Gleichung aufstellen:
  $$4x+4 = x^2 - 6x + 9$$
  $$0 = x^2 -10x + 5$$
  $$x_{1,2} = \frac{10 \pm \sqrt{100 - 20}}{2}$$
  $$\lx = \{0.52786..., 9.472\}$$
  Probe: Die Lösung 0.52786... ist eine Scheinlösung, denn die rechte
  Seite der ursprünglichen Gleichung würde negativ! Nur 9.472 ist Lösung.
}

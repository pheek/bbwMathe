\TNT{6}{Herleitung:
$A=\frac12\cdot{} c\cdot{}h_c$. Doch $h_c$ kann mit dem Sinus
berechnet werden: $$h_c  : b = \sin(\varepsilon) \Longrightarrow{}    h_c=b\cdot{}\sin(\varepsilon)$$
Somit ist $A=\frac12\cdot{}c\cdot{} (b\cdot{}\sin(\varepsilon))$.
}%% END TNT


\begin{gesetz}{Fläche allgemeines Dreieck}{}
  
Im allgemeinen Dreieck kann die Fläche $A$ wie folgt berechnet werden: 
$$A = \frac{1}{2}\cdot{}b\cdot{}c\cdot{}sin(\varepsilon)$$

«Die Hälfte von Seite mal Seite mal Sinus des Zwischenwinkels»

\end{gesetz}

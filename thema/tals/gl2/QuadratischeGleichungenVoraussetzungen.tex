\subsection{Voraussetzungen / Repetition}

A) Faktorisieren Sie:

$$x^2-12x+36$$

\TNT{6}{
  Binomische Formeln

  $$x^2-12x+36 = (x-6)\cdot{}(x-6)=(x-6)^2$$
}


B) Sie wissen, dass es Gleichungen mit mehreren Lösungen gibt:

$$|x-2| = 7$$

\TNTeop{a) $x-2 = 7$

  b) $x-2=-7 \Longrightarrow |x-2| = 7$

  $$\lx = \{-5; 9\}$$

}

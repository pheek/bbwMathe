%% Voraussetzungen fürs Verständnis der quadratischen Gleichungen

\subsection{Voraussetzungen / Repetition}

A) Sie wissen, dass es Gleichungen mit mehreren Lösungen gibt:

$$|x-2| = 7$$

\TNT{4}{a) $x-2 = 7$

  b) $x-2=-7 \Longrightarrow |x-2| = 7$

  $$\lx = \{-5; 9\}$$

}

B) Sie kennen Verfahren, um lineare Gleichungen zu lösen:

$$5x-b = ax + 3$$

\TNT{6}{
  Verfahren:\\1. $x$ separieren:
  $$5x-ax = 3+b$$
  2. $x$ ausklammern
  $$x(5-a) = 3+b$$
  3. durch die Klammern teilen
  $$x = \frac{3+b}{5-a}$$
  TRAINER: Bem. 1: $a\ne 0$

  Bem. 2: Alternativ: In Grundform bringen: $x=\frac{-b}a$
}
